BVH Tools released on the asset store

My BVH Tools package is now available on the Unity asset store for free too!

BVH Tools let you record and export motion data from avatars or skeletons to BVH files so they can be edited with Blender or other programs. The included animation loading component makes it possible import BVH files into Unity at runtime.

All components can be easily used from the editor with helpful tooltips for all fields and buttons to perform all necessary functions. Both the motion capture and animation loader components can be used from other scripts through a simple API, is documented in the readme file.

Animation import is efficient. For example, according to the Unity profiler, loading a 1.4 MB BVH file with slightly above 80 bones and 582 frames took slightly over 100ms during tests. Another file of almost 1.1 MB with 23 bones and 1522 frames took less than 75ms to load. A part of the loading process uses no Unity APIs, so it can be offloaded to a separate thread.

DokoDemoPainter released

You can now find my realtime texture painting package for Unity on GitHub.

It can paint on both regular mesh and skinned mesh. It detects objects to paint on using a camera and a special shader. Painting itself is also done using pixel shaders.

You can try out a very simple live demo here. A demo using WebVR can be found here. The WebVR demo is based on Mozilla's Sunny Desert demo.

VRM format

VRM is a format for 3D avatars. It's gaining more traction lately and the library allows run time importing of VRM models in Unity.

If you're interested in how to convert a model to VRM format, I wrote a short overview of the process.

Reassigning meshes to armatures

While working, I sometimes found myself wanting to attach a fresh mesh to an existing armature in Unity, because it already had a lot of components attached to various bones.

I found a script to do this on Unity answers, but it was still a little buggy.

After cleaning it up a bit, it works pretty well. You can find my version here.

Attach it to the new mesh, set the target armature and root bone and press the checkbox. It might take a while to run.
